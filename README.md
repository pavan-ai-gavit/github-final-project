# Simple Interest Calculator

## Project Description
A Bash-based calculator that computes **Simple Interest** based on user-provided inputs: principal amount, rate of interest, and time period.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Features
- Command-line interface using Bash
- User input for principal, rate, and time
- Instant simple interest calculation

## Usage
```bash
bash simple-interest.sh
```
